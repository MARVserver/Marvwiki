# Nation 国家運営ガイド

複数の街が集まって形成される「国家 (Nation)」についてのガイドです。

## 1. 国家を建てる
自分の街を首都として、国家を組織します。

- **建国コマンド**: `/nation create {国名}`
  - 多額の建国費用が必要ですが、広大な影響力を持つことができます。

## 2. 他の街を招待する
自分の国家に他の街を誘い、勢力を拡大します。

- **招待コマンド**: `/nation invite {街名}`
  - 招待された街の市長が同意すると、その街は国家の一員となります。

## 3. 外交 (Diplomacy)
他国との関係を定義します。

- **同盟**: `/nation ally {国名}`
  - 互いの領土で建築を許可したり、戦争時に協力したりできます。
- **敵対**: `/nation enemy {国名}`
  - 相手を敵としてマークします。

## 4. 維持費と税金
国家にも維持費がかかります。

- **入金コマンド**: `/nation deposit {金額}`
- **税金設定**: `/nation set taxes {金額}`
  - 加盟している各街から税金を徴収できます。

## 5. 国家の特典
- 独自のチャットチャンネル (`/nc`) が利用可能になります。
- 領地内でのボーナスや、より高度な保護設定が可能になります。
