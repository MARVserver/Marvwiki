# QuickShop-Hikari 使い方ガイド

MARVサーバーでは、チェストを使って自分のお店を開くことができます。

## 1. お店の作り方
1. チェスト（またはチェスト付きの乗り物等）を設置します。
2. 売りたいアイテムを手に持ちます。
3. チェストを右クリックします。
4. チャット欄に「1個あたりの価格」を入力して送信します。
5. 看板とアイテムのディスプレイが表示されれば作成完了です！

## 2. お店の管理
看板を左クリック（または右クリックでメニュー表示）することで、以下の操作が可能です。

### 主要コマンド
- `/qs create {単価}`: 手に持っているアイテムでショップを作成
- `/qs price {単価}`: 価格を変更
- `/qs buy`: 「買い取りモード」に切り替え（プレイヤーからアイテムを買い取る）
- `/qs sell`: 「販売モード」に切り替え（デフォルト）
- `/qs remove`: ショップを削除（チェストを壊しても削除されます）

### 便利な機能
- **コントロールパネル**: 看板を右クリックするとGUIが表示され、在庫の確認や設定変更が直感的に行えます。
- **店員設定**: `/qs staff add {プレイヤー}` で、他のプレイヤーに在庫の補充などを手伝ってもらうことができます。

## 3. 商品を探す
- `/qs find {アイテム名}`: 近くにある指定したアイテムのショップを探せます。
