# Towny 街運営ガイド

MARV Earthサーバーの中心となる「街 (Town)」の作成と運営についてのガイドです。

## 1. 街を作成する
自分の街を持つことで、土地を保護し、住人を募集することができます。

- **作成コマンド**: `/town create {街名}`
  - 作成には費用（サーバー設定による）が必要です。
  - 現在立っているチャンクが街の中心（ホームブロック）になります。

## 2. 領土を拡大する (Claim)
街の周囲のチャンクを確保して、土地を広げます。

- **確保コマンド**: `/town claim`
  - 自分の街に隣接するチャンクのみ確保可能です。
  - 街の人数や規模に応じて、確保できる上限数が増えていきます。

## 3. 運営と維持費 (Upkeep)
街を維持するには、毎日「サーバー維持費」がかかります。

- **入金コマンド**: `/town deposit {金額}`
  - 街の銀行（Town Bank）にお金を預けます。
  - 残高が不足すると、街が解散（フォール）してしまうので注意してください。
- **情報確認**: `/town` または `/town {街名}`

## 4. 住人を招待する
他のプレイヤーを住人として迎え入れましょう。

- **招待コマンド**: `/town invite {プレイヤー名}`
- **脱退コマンド**: `/town leave`

## 5. 土地の分譲 (Plot)
街の土地を区画分けして、住人に販売したり貸し出したりできます。

- **販売設定**: `/plot forsale {価格}`
- **購入（住人側）**: `/plot claim`
- **パーミッション設定**: `/plot set perm {on/off}`
