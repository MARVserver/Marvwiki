# プラグイン・コマンドガイド

MARVサーバーで利用できる主要なプラグインとそのコマンドについて解説します。

## 1. プレイヤーユーティリティ
サーバーでの生活を便利にするコマンド群です。

### スコアボード (marvtime)
- `/sbon`: スコアボードを表示
- `/sboff`: スコアボードを非表示

### 経験値管理 (Experience Bottle)
- `/bottle stats`: 現在のXPとレベルを表示
- `/bottle store {量} {個数}`: 指定したXPをボトルに詰める
- `/bottle mend`: 手に持っているツールの修理に必要な分だけボトルを作成
- `/bottle until {レベル}`: 指定レベルまでに必要なXPを表示

### エモート・モーション
- `/sit`: 座る
- `/lay`: 横になる
- `/spin`: くるくる回る
- `/crawl`: 匍匐前進

### 便利機能
- `/ec`: エンダーチェストを開く
- `/sethome {名前}` / `/home {名前}`: ホーム設定とワープ
- `/tfly`: 町の領土内での飛行（要権限・有効化）

---

## 2. 経済・ショップ
サーバー内での取引と資金管理に関するコマンドです。

### 紙幣・残高管理
- `/balance` / `/money`: 自分の残高を表示
- `/pay {プレイヤー} {金額}`: 他の人に送金
- `/withdraw {金額}`: 指定額の紙幣（アイテム）を発行
- `/deposit`: 手持ちの紙幣を換金（`stack` または `all` で一括換金可能）

### ショップ利用
- `/shop`: 公式メインショップを開く
- **QuickShop (チェストショップ)**: 
  - [詳細な使い方はこちら](#/ja/QuickShop)
- **職業システム (/jobs)**:
  - [詳細な稼ぎ方はこちら](#/ja/Jobs)

## 3. 保護・ロック権限
チェストやドアの保護を管理します。

- `/lock`: ブロックやエンティティをロック（自動保護）
- `/unlock`: ロックを解除
- `/bolt trust add {プレイヤー}`: 自分のすべての保護へのアクセスを許可
- `/bolt edit add {プレイヤー}`: 特定の保護ブロックに対してアクセスを許可

---

## 4. 組織 (Organization)
プレイヤー同士でグループ（会社・政党・軍など）を作成できます。

- `/org create {名前}`: 組織を作成
- `/org invite {プレイヤー}`: メンバーを招待
- `/org info {組織名}`: 組織の詳細情報を表示
- `/org setrole {プレイヤー} {役職}`: メンバーにカスタム役職を付与
